浙江省高考志愿投档数据分析、检索与预测系统

演示地址：https://zhejiang.helloworld2024.workers.dev/

db文件包含2017-2026的投档数据。

编译命令：

npm init -y

npm install vite vite-plugin-singlefile --save-dev

npm install bootstrap echarts xlsx

npm install sql.js

npm run build
